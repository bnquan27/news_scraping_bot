# What does this scraper do?

This scraper gets news from news.google.com and turns into json file format which includes informations such as: name, publishers, time, authors , link, and tags. Tags is generated by Gemini-api.

# Install

1. **Clone the repo and install dependencies:**
```shell
    git clone https://github.com/bnquan27/news_scraper
    cd news_scraper
    pip install -r requirements.txt
```
2. **Create a Google AI studio account and get your API key.** See the [Google Studio API key documentation](https://ai.google.dev/gemini-api/docs/api-key)

3. **Configure API key**
   Create a `.env` file in the `src/` directory (alongside `main.py`). Add your API key to this file like so:
   ```env
    GEMINI_API_KEY=YOUR_API_KEY_HERE
   ```
4. **Run the scraper**
    Navigate to the `src/` directory in your terminal and run:
    ```shell
    python main.py
    ```
    After the scraper run, you can see the list of news in the `articles.json` file. You can read the news by copy the link, paste into your browser and you are ready to go.
